Code : Mirror Number Pattern
Print the sample input pattern for the given N number of rows.

Input format :
Integer N (Total no. of rows)
Output format :
Pattern in N lines
Constraints
0 <= N <= 50

Sample Input 1:
3
Sample Output 1:
  1
 12
123

Sample Input 2:
4
Sample Output 2:
   1
  12
 123
1234
